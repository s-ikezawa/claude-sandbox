主要なパターンは以下の通り:
- パターン1: 同じコードの重複
- パターン2: マジックナンバー/マジックストリングの重複
- パターン3: 似たようなバリデーションロジックの重複
- パターン4: データ構造の重複定義
- パターン5: 条件分岐の重複
- パターン6: 設定値の重複
- パターン7: エラーハンドリングの重複

詳細は[DRY原則](dont-repeat-yourself.md)の本文を参照してください。
